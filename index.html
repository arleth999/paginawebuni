Introduccion a CSS3?

Esto fue posteado el 10/07/2018 por UMG-Jalapa

El nombre hojas de estilo en cascada viene del ingles Cascading Style Sheets, del que toma sus siglas. CSS es un lenguaje usado para definir la presentacion de un documento estructurado escrito en HTML o XML (y por extensión en XHTML). EL W3C(World Wide Consortium) es el encargado de formular la especificación de las hojas de estilo que servirán de estandar para los agentes de usuario o navegadores

Para que sirve CSS3?

El CSS sirve para definir la estetiva de un sitio web en un documento externo y eso mismo permite que modificando ese documento (la hoja CSS) podamos cambiar la estética entera de un sitio web, el mismo sitio web pude variar totalmene de estética cambiando solo el CSS, sin tocar para nada los documentos HTML o jsp o asp que lo componen
